# APIS Website

Premium static website for APIS - Advanced Pharma IT Services.

## Stack

- HTML
- Tailwind CSS via CDN
- Custom CSS
- Vanilla JavaScript

## Run Locally

Open `index.html` directly in a browser, or serve the folder with any static server.

```bash
python -m http.server 8080
```

## Notes

- Replace `https://example.com/` in meta tags, schema, `sitemap.xml`, and `robots.txt` with the production domain.
- WhatsApp inquiry links use dummy placeholder `xxxxx`.
- The hero uses a muted, looping optimized MP4 background at `assets/videos/dna-hero-web.mp4`.
